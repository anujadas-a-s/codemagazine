The Basic Language of Web:HTML

posted by Laura jone on Monday,june 21st 2017

All Modern website and web applications are built using three fundemetal technologies:HTML,CSS and Javascript.these are language of web.

In this post ,let's focus on HTML.We will learn what HTML is all about,and why you too should learn it.

what is HTML?

HTML stands for Hyper Text Markup Language.Its a markup language that Web developers use to structure and describe the content of webpage(not programming Language).

HTML consist of elements that describe discribe different types of content:paragraph,links,headings,images,videos,etc.Web browsers understand HTML and Render HTML code as website.
in HTML,each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
You can learn more at MDN Web Docs
